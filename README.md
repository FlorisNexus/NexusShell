# 🏗️ Nexus Command Center (v11.3)

> **The Neural Hub of the FlorisNexus Ecosystem.**

NexusShell is a professional, high-performance CLI dashboard built with **.NET 10** and **Spectre.Console**. It acts as an "AI-OS" to orchestrate multiple project tracks, automate marketing workflows, and maintain a unified architectural context across a global meta-workspace.

---

## 🌟 Core Philosophy: "The Strategic Dual-Track"

NexusShell is designed to manage two distinct business branches with zero friction:

### 🏗️ Branch 1: LOCAL HERO (Digital Presence)
- **Focus:** High-performance showcase sites for local SMEs and artisans.
- **Tech:** Vite + React + Tailwind CSS + Azure Static Web Apps.
- **Goal:** 100/100 Core Web Vitals and authentic regional SEO.

### 🌍 Branch 2: GLOBAL SAAS (International Tech)
- **Focus:** Scalable B2B/B2C products (Finova, Continuum).
- **Tech:** .NET 10 (Isolated Workers) + Blazor WASM + Azure PaaS.
- **Goal:** Robust, multi-tenant monitoring and financial management platforms.

---

## ⚡ Key Features

### 🏎️ Async Live UI Engine
A non-blocking, asynchronous interface with a 10ms response time. The hub performs heavy Git status scans and filesystem discovery in the background, ensuring navigation is always fluid.

### 🧠 Neural Links & Global Skills
Seamless integration with **Gemini CLI**. Launching a project track spawns a "Neural Link"—a scoped AI session that inherits **16+ Global Nexus Skills** (Clean Code, Performance, Infra, etc.) from the root workspace.

### 🏗️ Enterprise Scaffolding
Native C# implementation of project templates. Scaffold a production-ready SaaS API or a Local showcase site with integrated Bicep infrastructure and GitHub Action workflows in seconds.

### 🏁 Git & Sync Awareness
Real-time dashboard indicators for every project:
- **● Live Status:** Identifies which terminal windows are currently active.
- **↑2 ↓1 Sync:** Shows how many commits you are ahead or behind the remote.
- **! Changes:** Immediate visual warning for uncommitted local work.

### 📔 Founder's Journal & Marketing
Automated capture of daily insights and achievements, directly synced with an AI marketing strategist to draft authentic "Build in Public" stories for LinkedIn and X.

## 🚀 Getting Started

### Prerequisites
- .NET 10 SDK
- Gemini CLI
- Git

### Installation
1. Clone the repository into your `conductor` folder.
2. Build the hub:
   ```powershell
   cd conductor/NexusShell
   dotnet build -c Release
   ```
3. Add the launcher to your PowerShell `$PROFILE`:
   ```powershell
   function nexus { dotnet "C:\path\to\conductor\NexusShell\bin\Release\net10.0\NexusShell.dll" }
   ```
